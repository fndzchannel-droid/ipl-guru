# 🏏 IPLGURU — IPL 2026 Prediction Game

A fun match prediction app for friends & colleagues during IPL 2026!

## Features
- 📱 Register & Login via Mobile OTP
- 📅 Full IPL 2026 Schedule
- 🎯 Predict match winners
- 🏆 Live Leaderboard
- ⚙️ Admin Panel to update results (PIN: 1234)
- 📤 WhatsApp share for leaderboard & scores

## Getting Started

```bash
npm install
npm start
```

## Deploy to Vercel
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Import repo
3. Done! Your app is live 🚀

## Admin Access
- Click **Admin** tab in the nav
- Enter PIN: `1234`
- Update match scores & results after each game

## Demo OTP
Use `1234` as OTP for any mobile number during login/register.
